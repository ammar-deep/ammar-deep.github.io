**Soongsil University System Software Lab — 2018–2023**

A line of research applying generative models to font synthesis, character design, and image skeletonization.

**Few-Shot Font Generation:**

- Developed metric learning and contrastive learning network architectures for learning font style latent spaces
- Style-guided discriminator for improved style representation in few-shot settings
- Component-guided Korean and Chinese font generation algorithms
- Applications: text image editing, automated font library creation, cross-lingual font generation

**Font Family Synthesis:**

- Built an unpaired generative model for font family generation covering typeface variations
- Data collection, preprocessing, and labeling pipeline for font family datasets
- Patent granted (Korea, 2022): *Method and Apparatus for Generating Font Family Using Deep Learning*, Soongsil University Industry-Academic Cooperation Foundation, No. 2-2006-027849-9

**Text Image Skeletonization:**

- End-to-end GAN for character image skeletonization
- Skeleton-driven Korean font synthesis model (SKFont)
- Applications in object representation, manipulation, tracking, and recognition
